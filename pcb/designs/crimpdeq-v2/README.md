# Crimpdeq v2 Reference Design

Original two-layer Crimpdeq v2 KiCad source. This is the frozen electrical baseline used to
verify all derived designs; it is not the current compact fabrication target.

- Board: `crimpdeq.kicad_pcb`
- Schematic: `crimpdeq.kicad_sch`
- Project: `crimpdeq.kicad_pro`
- Nominal outline: 23.00 x 63.80 mm

Do not modify its BOM or connectivity while using it as the comparison baseline.
