# Crimpdeq Hardware

Hardware files and KiCad PCB design for Crimpdeq. See the [Crimpdeq book](https://crimpdeq.github.io/book/) for more details on the project!

## PCB

The PCB design is located under `hardware/crimpdeq/` and was created using KiCad. It is a two-layer board based on the [Rust ESP Board](https://github.com/esp-rs/esp-rust-board). This design removes unused sensors from the original board and adds the necessary components for this project.

![PCB](assets/pcb_v1.png)

The PCB was sponsored by [PCBWay](https://www.pcbway.com/), thank you! Working with them was incredibly easy and fast, and the resulting boards are high quality.

[![PCBWay](assets/PCBWay.png)](https://www.pcbway.com/)
